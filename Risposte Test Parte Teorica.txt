1) Descrivere le differenze tra Value e Reference Type
I value Type si trovano nello stack, hanno sempre un valore definito e sono disposti in modo ordinato e seguenziale.
Esempio di tipo valore è la struct a cui una chiave corrisponde un determinato valore.
Possono essere nullabili

I reference type si trovano nell'Heap quindi in un contesto meno ordinato dove non hanno un valore diretto ma puntano ad un oggetto. 
I reference sono sotto il controllo del Garbage Collector
Esempio di reference type è la classe
